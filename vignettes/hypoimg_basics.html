About the package:

The hypoimg package provides illustrations to annotate plots within hamlet studies (Hypoplectrus spp).

It contains the basic illustrations of different hamlet species and countries where hamlets occur. Additionally, the package provies a set of fuctions to construct legends which link a colormap with a set of hamlet species, pairwise species comparisons (or countries).

In addition to this the package contains a few helpers for a consistent plot layout. More artistic aspects of hamlet analysis plots are provided by the hypoimg package.

Installation

Using the devtools package, hypoinmg can be installed directly from github:

devtools::install_github("k-hench/hypoimg")
